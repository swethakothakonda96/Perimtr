# Perimetr
Perimtr is a browser based attendance and polling platform built to guarantee Proof of Presence in classrooms, corporate trainings, and high stakes meetings. Unlike traditional tools that rely on easily shareable links, Perimtr ensures that every attendance check in, vote, or poll response comes from a real person physically present in the room.
By combining local network verification, device level identity, and time bound credentials, Perimtr eliminates attendance fraud, proxy voting, and duplicate submissions, without requiring users to download an app or create an account.
Core Concept: Proof of Presence
Perimtr enforces three foundational principles:
1.	Same Network Access Only
Participants can join a session or poll only if they are connected to the same Wi Fi network as the administrator. Remote users are automatically blocked with an “Access Denied” message.
2.	One Device = One Person
Each device receives a unique hardware based token, preventing:
•	Multiple votes
•	Duplicate attendance
•	Browser switching or incognito bypassing
3.	Zero Friction for Participants
No logins, no downloads, no setup. Users simply scan a QR code, enter their name and PIN, and participate instantly.

This application is completely built through prompts. The initial working platform was built with replit. Consecutive changes are done with Cursor and Antigravity
